🔮 ElectIQ

AI Election Forecasting Bot

Real-time predictive modeling for global political elections

The Challenge

Modern elections are driven by overwhelming amounts of data:

  • Information Overload: News cycles update by the minute.
  • Stale Polling: Traditional polls lag behind fast-moving digital sentiment.
  • Complex Demographics: Shifting voter bases require deep, multi-variable analysis.

The Solution: ElectIQ

ElectIQ deploys a sophisticated Neural Swarm of AI agents.

  • Analyzes live web data and historical trends instantly.
  • Evaluates regional sentiment and demographic shifts.
  • Provides real-time predictive modeling with quantitative probability scores.

The Telemetry Stream

Instead of making users wait 15+ seconds for a massive JSON payload...

The server pipes the AI's internal "thought process" back to the client chunk-by-chunk using Server-Sent Events (SSE).

Result: A "hacker-style" live telemetry terminal that engages users instantly.

🧠 Primary AI Engine

Groq API (llama-3.3-70b-versatile)

  • Powered by Groq's LPU™ Inference Engine for blazing-fast token generation.
  • Handles complex reasoning and historical comparisons.
  • Streams insights directly to the user interface with near-zero latency.

🛡️ Fallback AI Engine

Google Gemini (gemini-2.5-flash)

  • Seamlessly takes over if the primary engine hits rate limits.
  • Utilizes Google Search Grounding.
  • Injects real-time web data and breaking news directly into the forecast.

Interactive UI Components

  • PartyLogo: Dynamically queries the Wikipedia API to fetch and render high-resolution SVG political party logos.
  • ProbabilityGauge: Custom SVG visualizations using Recharts displaying win probability as glowing circular gauges.
  • SentimentMeter: Visual bars indicating positive/negative public sentiment based on AI scraping.
  • AIExplanation: Renders qualitative data including decisive factors and risk vectors.

⚙️ Backend Security & Mechanics

  • Concurrency Mutex: A global lock ensuring only one heavy AI forecast runs simultaneously, protecting against DDoS and API exhaustion.
  • Regex Validation: Strict pre-flight checks ensure queries are explicitly political before hitting LLM APIs.
  • In-Memory Cache: 60-minute TTL cache returns instant forecasts for recurring queries.
  • Deep Diagnostics: Custom logging captures uncaught exceptions, stream events, and fallback triggers.
